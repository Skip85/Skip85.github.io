PR-проект для г.Мичуринск

Цели

1. Внедрить информационную систему;

2. Заинтересовать население в общественной жизни города;

3. Сформировать благоприятный имидж власти.

Задачи

1. Укрепление сотрудничества общества и органов местного управления;

2. Обеспечение информационной прозрачности и открытости органов местного управления для жителей города;

3. Расширение спектра и улучшение качества предоставляемых гражданам государственных услуг;

4. Оптимизация деятельности государственных органов.

Реализация

Этапы реализации проекта представляют собой повторяющийся несколько раз цикл определенных планом работ, связанных с реализацией мероприятий плана, измерением и оцениванием результатов, коррекции плана с целью повышения эффективности результата и оценку полученных результатов. Цикл повторяется до достижения требуемого результата при выполнении условий по бюджету проекта. Методики измерения результатов основаны на изучении активности СМИ по PR-проекту и отзывов методами контент-анализа, реакцией целевых групп на форумах и в социальных сетях, другими косвенными методами измерения результатов. Возможны и прямые методы измерения результатов с использованием опросных листов и анкетирования.

Этапы работ по внедрению системы
  • 1 Шаг

    Разработать типовой вариант автоматизированной муниципальной услуги населению

  • 2 Шаг

    Создание Центра поддержки автоматизированной системы управления муниципальной услуги

  • 3 Шаг

    Объединение проекта с проектом «Электронная Россия»
